🔹 STEP 5: Database Design
4
🗄️ Tables
1. Users Table
Field	Type
id	INT (PK)
name	VARCHAR
email	VARCHAR
password	VARCHAR
role	ENUM(user, staff, worker, admin)
phone	VARCHAR
2. Stations Table
Field	Type
id	INT
name	VARCHAR
location	VARCHAR
total_slots	INT
available_slots	INT
staff_id	INT
3. Slots Table

| id | station_id | slot_time | status |

4. Bookings Table
Field	Type
id	INT
user_id	INT
station_id	INT
slot_id	INT
date	DATE
status	ENUM(pending, approved, rejected)
qr_code	TEXT
5. Workers Table

| id | user_id | assigned_station |

6. Complaints Table

| id | user_id | message | status |
